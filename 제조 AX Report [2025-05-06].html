The automated framework 'LASSI-EE', which optimizes the energy efficiency of parallel scientific code using LLMs, has been released. The NVIDIA A100 GPU test results showed an average energy saving effect of 47% in 85% of the code, which is amazing. 😲

Implications: AI technology shows that it can contribute to the important value of 'sustainability' beyond simply improving performance.
